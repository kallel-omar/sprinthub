# SprintHub

A modern Project Management SaaS built with Symfony 7.

## 🚀 Features

### Current Features

* User Registration
* User Authentication
* Dashboard
* Workspaces
* Projects
* Responsive Bootstrap UI

### Planned Features

* Tasks
* Kanban Boards
* Team Collaboration
* Comments
* Notifications
* Activity Logs
* Roles & Permissions

## 🛠 Tech Stack

* Symfony 7
* PHP 8.2
* Doctrine ORM
* MySQL
* Twig
* Bootstrap 5

## 📂 Architecture

User
→ Workspace
→ Project
→ Task

## Installation

```bash
git clone https://github.com/kallel-omar/sprinthub.git
cd sprinthub
composer install
symfony server:start
```
